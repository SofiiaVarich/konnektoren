# konnektoren

Konnektoren

## Quickstart

## git clone and pull

```sh
git pull
```

### Prerequisites

```bash
cargo install trunk wasm-bindgen-cli
```

### Build

```bash
cargo build
```

### Run

```bash
trunk serve
```

### Run TUI on terminal

```bash
cargo run --bin konnektoren-tui --features=tui
```
